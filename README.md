## Run Locally

Download 

- node
- npm

Clone the project 

```bash
git clone https://github.com/iSuyash69/ScanEat-Order-Simplified
```

#### Client side  :

Change to Project Directory

Install Dependencies

```bash
  npm install -D parcel
```
```bash
  npm install
```

Start React App

```bash
  npm run start
```

#### Server side :

Change to Project Directory 

```bash
  cd server
```

Install Dependencies

```bash
 npm install
```

Start Server

```bash
  node index.js
```
